HTML元素、属性

HTML 元素指的是从开始标签"<"到结束标签"/>"的所有代码。

例如文本段落标签是以<p>开头，<p>结尾的。

属性通常在开始标签里书写

例如<p style="color:red">

属性通常可以在元素后面书写，但是我不建议这样，原因在之后的CSS上会告诉你。

HTML 属性

属性通常以 名称="值" 的形式出现，比如：name="value"。

HTML标签有属性，属性提供了HTML元素的更多的功能。（例如刚刚的内容变色）

更多例子：

<body>的背景颜色被修改成了"indianred"，<h2>的内容居中。

<br>：换行，页面效果就是换行。

<hr>：水平线，页面效果就是一条水平线。

<!-- -->：注释，注释内容包含在开始结束标签里面"<!-- 这里是注释内容 -->"

更多属性可在相关资料网站找到，我会告诉你一个页面怎么形成，你可以明白其代码书写用途。但不包含了解所有代码功能，需要自己查找补课。

空的 HTML 元素

没有内容的 HTML 元素被称为空元素。空元素是在开始标签中关闭的。

<br> 就是没有关闭标签的空元素（<br> 标签在页面的效果是换行）。

在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须被关闭。

在开始标签中添加斜杠，比如 <br />，是关闭空元素的正确方法，HTML、XHTML 和 XML 都接受这种方式。

即使 <br> 在所有浏览器中都是有效的，但使用 <br /> 其实是更长远的保障。

如果你不理解空元素，那么可以简单认为其没有结束标签，通常在开始标签书写语句。
